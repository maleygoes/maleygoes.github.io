    Section 5.6 Linear Functions Review Questions

    Subsection 5.6.1 Slope

    Exercises Exercises

    1.
    \(g(x)\) is a linear function with \(g(-1) = 5\) and \(g(4) = -6\text{.}\) Find its slope.
    🔗
    Solution.
    \(-2.2\)
    🔗
    🔗
    🔗
    2.
    \(f(x)\) is a linear function with \(f(-5) = -14\) and \(f(1) = -2\text{.}\) Find its slope.
    🔗
    Solution.
    \(2\)
    🔗
    🔗
    🔗
    3.
    \(g(t)\) is a linear function with \(g(2) = -19\) and \(g(8) = -49\text{.}\) Find its slope.
    🔗
    Solution.
    \(-5\)
    🔗
    🔗
    🔗
    4.
    Suppose \(k(m)\) is linear. Use the table below to find its slope.
    \begin{equation*} \begin{array}{c|c} m \amp k(m) \\ \hline -7 \amp 3,000 \\ 10 \amp -400 \\ 250 \amp -48,400 \end{array} \end{equation*}
    🔗
    Solution.
    \(-200\)
    🔗
    🔗
    🔗
    5.
    Given the table below, find the slope of the linear function.
    \begin{equation*} \begin{array}{c|c} x \amp f(x) \\ \hline -2 \amp 6 \\ 2 \amp -4 \\ 7 \amp -16.5 \\ 10 \amp -24 \end{array} \end{equation*}
    🔗
    Solution.
    \(-2.5\)
    🔗
    🔗
    🔗
    6.
    Find the slope of the linear function whose graph passes through the points \((4,1)\) and \((10, -8)\text{.}\)
    🔗
    Solution.
    \(-1.5\)
    🔗
    🔗
    🔗
    7.
    Find the slope of the linear function whose graph passes through the points \((0, -12)\) and \((14, -8.5)\text{.}\)
    🔗
    Solution.
    \(0.25\)
    🔗
    🔗
    🔗
    🔗
    🔗

    Subsection 5.6.2 Intercepts

    Exercises Exercises

    1.
    Determine the vertical and horizontal intercepts for each of the linear functions.
    1. \(\displaystyle g(t) = -5t + 20\)
      🔗
      🔗
    2. \(\displaystyle f(q) = 16 - \frac{3}{4}q\)
      🔗
      🔗
    3. \(\displaystyle h(k) = -1.2(k - 3) + 4\)
      🔗
      🔗
    4. \(\displaystyle j = \frac{2x}{5}\)
      🔗
      🔗
    5. \(\displaystyle h(t) = -2t + 4\)
      🔗
      🔗
    6. \(\displaystyle g(n) = 2n - 7\)
      🔗
      🔗
    7. \(\displaystyle f(b) = \frac{1}{2}b + \frac{7}{3}\)
      🔗
      🔗
    🔗
    Solution.
    a. Vertical: \((0,20)\text{,}\) Horizontal: \((4,0)\) \b. Vertical: \((0,16)\text{,}\) Horizontal: \((\frac{64}{3},0)=(21.33,0)\) c. Vertical: \((0,7.6)\text{,}\) Horizontal: \((\frac{19}{3},0)=(6.33,0)\) d. Vertical: \((0,0)\text{,}\) Horizontal: \((0,0)\) e. Vertical: \((0,-7)\text{,}\) Horizontal: \((3.5,0)\) f. Vertical: \((0,\frac{7}{3})=(0, 2.33)\text{,}\) Horizontal: \((-\frac{14}{3},0)=(-4.67,0)\)
    🔗
    🔗
    🔗
    2.
    If the function is linear, state the slope and vertical intercept.
    1. \(\displaystyle a(b) = 34 - 50b\)
      🔗
      🔗
    2. \(\displaystyle c = 10 - 2(x + 4)\)
      🔗
      🔗
    3. \(\displaystyle d(f) = 15(f - 2) + 4(2f + 6)\)
      🔗
      🔗
    4. \(\displaystyle h(x) = 2(x - 2) + x(x + 4)\)
      🔗
      🔗
    🔗
    Solution.
    a. Slope: \(-50\text{,}\) Vertical: \((0,34)\) b. Slope: \(-2\text{,}\) Vertical: \((0,2)\) c. Slope: \(23\text{,}\) Vertical: \((0,-6)\) d. Not linear
    🔗
    🔗
    🔗
    3.
    The number of bags of fertilizer needed to fertilize a lawn of size \(A\) acres is given by the function \(f(A) = 11A\text{.}\) Write a sentence interpreting the vertical intercept and slope.
    🔗
    Solution.
    Slope: The number of bags of fertilizer needed increases by \(11\) bags per acre. Vertical: No bags of fertilizer are needed for zero acres.
    🔗
    🔗
    🔗
    4.
    The monthly water bill for a resident of the town of Williamson is given by the function \(w(g) = 25 + 0.005g\text{.}\) Write a sentence interpreting the vertical intercept and slope.
    🔗
    Solution.
    Slope: The residents of the town of Williamson are charged an additional \($0.005\) per additional gallon of water used each month. Vertical: The residents of the town of Williamson are charged a monthly fee of \($25\) for water.
    🔗
    🔗
    🔗
    5.
    The cost for a cab ride is a function of the distance traveled. It costs \(25.50\) to travel \(15\) miles and \(34.50\) to travel \(21\) miles. Find the slope and vertical intercept and interpret both given the context of this function.
    🔗
    Solution.
    Slope: The cost for a cab ride increase by\($1.50\) for each additional mile. Vertical: It costs \($3.00\) to have the cab pick you up.
    🔗
    🔗
    🔗
    6.
    For the question above Exercise 5.6.2.5, find the horizontal intercept. Does that have a realistic meaning here?
    🔗
    Solution.
    \((-2,0)\text{;}\) no
    🔗
    🔗
    🔗
    🔗

    Subsection 5.6.3 Formula

    Exercises Exercises

    1.
    Find the equation of a linear function whose slope is \(-\frac{2}{9}\) with a vertical intercept of \((0,-5)\text{.}\)
    🔗
    Solution.
    \(y=-\frac{2}{9}x-5\)
    🔗
    🔗
    🔗
    2.
    Find the formula of a linear function whose slope is \(5\) and its graph passes through the point \(f(6) = 15\text{.}\)
    🔗
    Solution.
    \(y=5x-15\)
    🔗
    🔗
    🔗
    3.
    Suppose \(h(x)\) is linear, and its graph passes through the points \((-5, 0.2)\) and \((3, 10)\text{.}\) Find a formula for this function.
    🔗
    Solution.
    \(y=1.225x+6.325\)
    🔗
    🔗
    🔗
    4.
    Find a formula for the linear function whose slope is \(6\) and passes through the point \((0, -10)\text{.}\)
    🔗
    Solution.
    \(y=6x-10\)
    🔗
    🔗
    🔗
    5.
    Find a formula for the linear function whose graph passes through the points \((-3,24)\) and \((3,0)\text{.}\)
    🔗
    Solution.
    \(y=-4x+12\)
    🔗
    🔗
    🔗
    6.
    Given that \(w(t)\) is linear and, \(w(-5) = -85\) and \(w(10) = 95\text{,}\) find a formula for this function.
    🔗
    Solution.
    \(w(t)=12t-25\)
    🔗
    🔗
    🔗
    7.
    The price of a first class stamp in 1971 was \(0.07\text{.}\) By 2018 it rose to \(0.50\text{.}\) Assuming the price of a stamp is a linear function of years since 1971, write an equation that models this relationship.
    🔗
    Solution.
    \(y=0.009t+0.07\)
    🔗
    🔗
    🔗
    8.
    The cost to refinish a hardwood floor is a linear function of the size of the room, in square feet. It costs \($700\) to refinish a room \(100\) sq. feet. And \($950\) for a room \(150\) sq. feet. Find a formula for this linear function.
    🔗
    Solution.
    \(r(s)=5s+200\)
    🔗
    🔗
