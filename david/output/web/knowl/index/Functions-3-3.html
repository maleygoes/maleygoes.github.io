Definition 4.1.1.

Function Definition
🔗
A function is a rule which processes information by assigning one and only one output to each input. If the input variable’s name is \(x\text{,}\) the output variable is \(y\text{,}\) and the name of the function is \(f\text{,}\) we write \(y=f(x)\) which is read as “\(y\) equals \(f\) of \(x\text{.}\)” (Other letters can be used for any of the quantities involved, but whatever letters are used they follow the same pattern.)
🔗
Functions can be represented using a function diagram like this:
\begin{equation*} input \rightarrow f \rightarrow output \end{equation*}
or using variables:
\begin{equation*} x \rightarrow f \rightarrow y \text{ OR } x \rightarrow f \rightarrow f(x) \end{equation*}
🔗
When a function exists between two quantities, we say that “the output variable (or quantity) is a function of the input variable (or quantity).”
