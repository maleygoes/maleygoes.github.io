A system of linear equations, or linear system , is simply any collection of one or more linear equations. For example:
\begin{equation*} \begin{cases}2x+3y=12\\5x-4y=30 \end{cases} \end{equation*}
is a linear system because it consists of these two linear equations. The bracket that you see on the left side is often used with a linear system to make it clear that these equations are meant to be considered together as a linear system, and not just as two individual equations with no connection to each other.
