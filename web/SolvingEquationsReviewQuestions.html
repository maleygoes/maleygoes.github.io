    Subsection 3.5.1 Linear Equations

    Determine which, if any, of the given values satisfy the given equations.

    Exercises Exercises

    1.
    \(2a - 7 = 30; a = 13, -11, \text{ or } 7\)
    Solution.
    none
    2.
    \(18 - 2b = 0; b = -4, 5, \text{ or } 9\)
    Solution.
    \(b=9\)
    3.
    \(c^2 - c = 12; c = -4, -3, \text{ or } 4\)
    Solution.
    \(c=-3\) or \(c=4\)
    4.
    \(2x + 4 = 2; x = -5, -1, 0, \text{ or } 2\)
    Solution.
    \(x=-1\)
    5.
    \(2x^2 + 1 = 19; x = -6, -4, -3, 0, 2, \text{ or } 3\)
    Solution.
    \(x=3\) or \(x=-3\)

    State whether the equation is an identity, contradiction, or neither.

    Exercises Exercises

    1.
    \(15r- 45 = 5(-9 + 3r)\)
    Solution.
    identity
    2.
    \(4(5s - 6) = 32 + 20s\)
    Solution.
    contradiction

    \(-7t + 2(3t - 1) = 4 - t\)
    Solution.
    contradiction
    Solve each of the following equations for the requested variable in terms of the other variable(s).

    Exercises Exercises

    1.
    \(2x - y = 9\text{;}\) solve for \(y\)
    Solution.
    \(y=\frac{9-2x}{-1}=-9+2x\)
    2.
    \(5g + 9h = -8\text{;}\) solve for \(h\)
    Solution.
    \(h=\frac{-8-5g}{9}\)
    3.
    \(5g + 9h = -8\text{;}\) solve for \(g\)
    Solution.
    \(g=\frac{-8-9h}{5}\)
    4.
    \(\frac{k-3}{4} = j\text{;}\) solve for \(k\)
    Solution.
    \(k=4j+3\)
    5.
    \(\frac{2}{7}m + 1 = n\text{;}\) solve for \(m\)
    Solution.
    \(m=\frac{7}{2}(n-1)\)
    6.
    \(h = 3g + 4\text{;}\) solve for \(g\)
    Solution.
    \(g=\frac{h-4}{3}\)
    7.
    \(t = -2s - 1\text{;}\) solve for \(s\)
    Solution.
    \(s=\frac{t+1}{-2}=-\frac{1}{2}t-\frac{1}{2}\)
    8.
    \(g = \frac{k+2}{5}\text{;}\) solve for \(k\)
    Solution.
    \(k=5g-2\)
    9.
    \(d = abc\text{;}\) solve for \(b\)
    Solution.
    \(b=\frac{d}{ac}\)
    10.
    \(w = \frac{xy}{z}\text{;}\) solve for z
    Solution.
    \(y=\frac{wz}{x}\)

    Subsection 3.5.2 Second Degree Equations

    Solve each of the following for the given variable.

    Exercises Exercises

    1.
    \(a^2 + 7a = -12\)
    Solution.
    \(a=-4\) or \(a=3\)
    2.
    \(2b^2 - 72 = 0\)
    Solution.
    \(b=-6\) or \(b=6\)
    3.
    \(-77 = -c^2 - 4c\)
    Solution.
    \(c=-11\) or \(c=7\)
    4.
    \(d^2 - 25 = 0\)
    Solution.
    \(d=-5\) or \(d=5\)
    5.
    \(2f^2 - 6f - 80 = 0\)
    Solution.
    \(f=-5\) or \(f=8\)
    6.
    \((3g - 1)(2g + 7) = 0\)
    Solution.
    \(g=\frac{1}{3}\) or \(g=-\frac{7}{2}\)
    7.
    \((h + 4)^2 - 3 = 22\)
    Solution.
    \(h=-9\) or \(h=1\)
    8.
    \(3(j - 6)^2 + 10 = 37\)
    Solution.
    \(j=3\) or \(j=9\)
    9.
    \(2p^2 - 8 = 0\)
    Solution.
    \(p=-2\) or \(p=2\)
    10.
    \((k + 4)(k - 4) = 0\)
    Solution.
    \(k=-4\) or \(k=4\)
    11.
    \(r^2 + r - 42 = 0\)
    Solution.
    \(r=-7\) or \(r=6\)
    12.
    \(g^2 - 64 = 0\)
    Solution.
    \(g=-8\) or \(g=8\)

    \(u^3 = 125\)
    Solution.
    \(u=5\)
    2.
    \(-s^3 = 64\)
    Solution.

    3.
    \(v^3 + 7 = 34\)
    Solution.

    Solution.
    \(z=\sqrt[3]{-5} \approx -1.71\)
