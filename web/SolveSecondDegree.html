    Section 3.2 Solving Second Degree Equations

    Subsection 3.2.1 The Square Root Property

    In order to solve a second-degree equation, or an equation with an \(x^2\) term, we have to get rid of the exponent to be able to solve for \(x\text{.}\) In many second-degree equations, we can do this simply by taking the square root of both sides of the equation. For example, to solve the following:
    \begin{equation*} x^2=25 \end{equation*}
    🔗
    We could undo the squaring of \(x\) by taking the square root of both sides. There is, though, a wrinkle in this approach. There are actually two solutions to this equation: \(x\) could be \(5\) since \(5^2=25\text{,}\) but \(x\) could also be \(-5\) because \((-5)^2=25\) as well. The square root function, which is usually denoted using the radical symbol “\(\sqrt{}\)”, is interpreted to mean just the positive square root. Yet the positive square root is not the only solution to the equation, and so we need to make sure we don’t loose the other solution in the algebra. To do this we follow the square root property which tells us that we can take the square root of both sides of the equation so long as we acknowledge that there are two possible square roots, the positive and the negative. We usually do this using the symbol “\(\pm\)” which is read as “plus or minus”. So the next step in solving \(x^2=25\) would be:
    \begin{equation*} x=\pm \sqrt{25} \end{equation*}
    leading to the solutions:
    \begin{equation*} x=\pm 5 \end{equation*}
    🔗
    It is important to understand that this notation is actually telling us that there are two solutions to the equation. There is no such number as \(\pm 5\text{.}\) This notation is just a compact way of expressing that there are to the equation \(x^2=25\text{,}\) specifically it is compact notation for the expression:
    \begin{equation*} x=5 \text{ or } x=-5 \end{equation*}
    🔗
    This method can be applied to other equations, as the following example will illustrate:
    🔗

    Example 3.2.1.

    Solve \(4x^2-25=11\)
    🔗
    🔗
    To apply the square root property we need to solve for \(x^2\) as if it were simply a variable. Thus, we start by adding \(25\) to both sides of the equation to obtain:
    \begin{equation*} 4x^2=36 \end{equation*}
    and then we divide both sides by \(4\) so that:
    \begin{equation*} x^2=9\text{.} \end{equation*}
    🔗
    Applying the square root property now gives:
    \begin{equation*} x=\pm \sqrt{9} \end{equation*}
    \begin{equation*} x=\pm 3 \end{equation*}
    🔗

    Example 3.2.2.

    Solve \(2x^2+7=39\)
    🔗
    🔗
    We start by subtracting \(7\) from both sides:
    \begin{equation*} 2x^2=32 \end{equation*}
    🔗
    To get \(x^2\) by itself, we divide by \(2\text{:}\)
    \begin{equation*} x^2=16 \end{equation*}
    🔗
    Applying the square root property now gives:
    \begin{equation*} x=\pm \sqrt{16} \end{equation*}
    \begin{equation*} x=\pm 4 \end{equation*}
    🔗

    Example 3.2.3.

    Solve \(3z^2-9=z^2+6\)
    🔗
    🔗
    To do this we first use ordinary algebra to solve for \(z^2\text{,}\) so subtract \(z^2\) from both sides:
    \begin{equation*} 2z^2-9=6 \end{equation*}
    and then add \(9\) to both sides:
    \begin{equation*} 2z^2=15 \end{equation*}
    🔗
    Now to get \(z^2\) by itself we divide by 2:
    \begin{equation*} z^2=\frac{15}{2} \end{equation*}
    🔗
    Now we are ready to use the square root property. Doing so we end up with two solutions
    \begin{equation*} z=\pm \sqrt{\frac{15}{2}} \end{equation*}
    🔗
    Since neither \(15\) nor \(2\) is a perfect square we are stuck on simplifying this any further. This is a perfectly fine way to express the answer. If, however, you would like to have a decimal approximation. it can be obtained using your calculator as \(z=\pm 2.7386\text{....}\) The problem with converting to a decimal is that it does typically require rounding so it is often preferable to leave your answer in radical notation as that is the exact answer and can be rounded to any level of precision later on. }
    🔗
    🔗

    Subsection 3.2.2 Factor to Solve a Second-Degree Equation

    In some cases, a second degree equation may be written in a way that makes the solutions very easy to find. For example:
    \begin{equation*} (x-4)(x+1)=0 \end{equation*}
    🔗
    We know that can be simplified by using the distributive property so we know this is indeed a second degree equation.
    🔗
    We know from basic arithmetic that if you multiply two numbers together and get zero, one of these numbers must be zero. Formally, this fact is called the zero product property.
    🔗

    Definition 3.2.4.

    Zero Product Principle
    🔗
    If \(AB=0\text{,}\) then either \(A=0\) or \(B=0\text{.}\)
    🔗
    🔗
    Now, in this equation we have two expressions being multiplied together: \(x-4\) and \(x+1\text{.}\) The equation says that their product is zero. So, according to the zero product property, the only possibilities are
    \begin{equation*} x-4=0 \text{ or } x+1=0 \end{equation*}
    🔗
    Using this fact we know there are two possibilities to consider. One is that \(x-4=0\text{.}\) If that is the case, then we can easily solve and see that \(x=4\text{.}\) The other possibility is that \(x+1=0\text{.}\) In that case, we can also easily solve and see that \(x=-1\text{.}\) So, we conclude that this equation’s solutions are \(x=4\) and \(x=-1\text{.}\) When a second degree equation is written in a form like this, as a product, we can find the solutions simply by setting each of the factors equal to zero and solving. Here is another example:
    🔗

    Example 3.2.5.

    Solve: \(5(2x-3)(x+7)=0\)
    🔗
    🔗
    To solve \(5(2x-3)(x+7)=0\) we set each factor equal to zero and determine if that leads to a consistent equation. Since the product is zero, we can apply the zero product principle to conclude that either \(5=0\) or \(2x-3=0\) or \(x+7=0\text{.}\) Since the first factor is a number and it is not equal to zero, we get a contradiction as \(5 \ne 0\text{.}\) Thus, the first factor does not lead to a solution to the equation in question. The second factor, \(2x-3\) is zero when \(x=\frac{3}{2}\text{.}\) The third factor \(x+7\) is zero when \(x=-7\text{.}\) Thus, there are two solutions to the equation \(x=\frac{3}{2}\) and \(x=-7\text{.}\)
    🔗
    As we saw in section 2.6 this second-degree equation is written as the product of expressions, or we can say in factored form. In some cases, we can use the method given to us in section 2.6 to factor the second degree expression to help us solve. To review:
    🔗

    Definition 3.2.6.

    Factoring Second Degree Expressions of the Form \(x^2+bx+c\)
    1. \(\cdot\).
      Consider all the pairs of integers whose product is \(c\)
      🔗
      🔗
    2. \(\cdot\).
      If the sum of any of the pairs is \(b\text{,}\) then the expression can be factored to \((x+m)(x+n)\) where \(m\) and \(n\) are the integers.
      🔗
      🔗
    3. \(\cdot\).
      If none of the pairs sum to \(b\text{,}\) then the expression cannot be factored using integers. We then say is it “not factorable over the integers”.
      🔗
      🔗
    🔗
    When groupings of the same variable raised to power or exponents, you can just add the exponents
    🔗
    (make sure to remember that a ‘plain’ \(x\) is actually \(x^1\)).
    \begin{equation*} x^a \cdot x^b = x^{a+b} \end{equation*}
    🔗
    🔗

    Example 3.2.7.

    Factor \(x^2-5x+6\)
    🔗
    🔗
    We need to consider all possible pairs of integers whose product is \(6\) and then look at their sum to see if it is possible to obtain a sum of the factors equal to \(-5\text{.}\) We consider all possible pairs \(1\) and \(6\) (sum \(7\)), \(-1\) and \(-6\) (sum \(-7\)), \(2\) and \(3\) (sum \(5\)), and \(-2\) and \(-3\) (sum \(-5\)). The last pair of factors are the factors that we want and \(x^2-5x+6=(x-2)(x-3)\)
    🔗
    In this case, there were not many possibilities to consider, and one of those possibilities worked. In many cases, though, there are either too many possible pairs to consider, or the expression can’t be factored over the integers at all. In the special case where the expression can be readily factored, solving by factoring is a reasonable method to use.
    🔗

    Example 3.2.8.

    Solve \(x^2-5x+6=0\) using factoring
    🔗
    🔗
    Using example 3.2.7, we can factor the left side of the equation as \((x-2)(x-3)=0\text{.}\) Applying the zero product principle we see that the solutions to the equation come from \(x-2=0\) or \(x-3=0\text{.}\) Thus, the solutions we are looking for are \(x=2\) and \(x=3\text{.}\)
    🔗

    Example 3.2.9.

    Solve \(x^2+8x+7=0\) using factoring
    🔗
    🔗
    There are not too many possibilities to consider here, since \(7\) is prime, and so the only pairs to consider are \(7\) and \(1\) or \(-7\) and \(-1\text{.}\) Since \(7 + 1 = 8\) we can factor this expression in \((x+1)(x+7)\) so we need to consider:
    \begin{equation*} (x+1)(x+7)=0 \end{equation*}
    🔗
    Setting each factor equal to zero, we solve \((x+1)(x+7)=0\text{.}\) So either \(x+1=0\) in which case \(x=-1\) or \(x+7=0\) in which case \(x=-7\text{.}\) So the solutions are \(x=-1\) and \(x=-7\text{.}\) }
    🔗

    Example 3.2.10.

    Solve \(x^2-9x+20=0\) using factoring
    🔗
    🔗
