    Section 2.2 Distribution

    The distributive law, one of the basic laws of arithmetic and algebra, states that when you have something multiplied by an expression in parentheses, you can multiply it by each of the terms inside the parentheses.
    🔗
    Well, that was awfully wordy. It’s far easier to represent this using symbols:
    \begin{equation*} a(b+c)=ab+ac \end{equation*}
    where \(a\text{,}\) \(b\) and \(c\) represent any algebraic expressions at all.
    🔗
    The distributive law is often put to use in simplifying algebraic expressions, as the next few examples will illustrate.
    🔗

    Example 2.2.1.

    Distribute and then simplify the expression \(5(2x+3y)\text{.}\)
    🔗
    🔗
    Following the distributive law, we can multiply 5 by each term inside the parentheses:
    \begin{equation*} 5(2x+3y)=5(2x)+5(3y)=10x+15y \end{equation*}
    🔗
    The instructions for example 2.2.1 explicitly told us to distribute. Usually, though, it is understood that distributing whenever possible is a part of simplifying, and so you should assume that distributing is expected when you are asked to simplify an expression as in the next example.
    🔗

    Example 2.2.2.

    Simplify \(5x(2x^2+3x+1)+3x(2x-5)\text{.}\)
    🔗
    🔗
    We’ll distribute first, and then simplify each of the resulting terms. Finally, we will combine like terms.
    \begin{align*} 5x(2x^2+3x+1)+3x(2x-5) \amp = 5x(2x^2)+5x(3x)+5x(1))+3x(2x)+3x(-5)\\ \amp\\ \amp = 10x^3+15x^2+5x+6x^2+-15x\\ \amp\\ \amp = 10x^3 + 21x^2 -10x \end{align*}
    🔗
    Usually we don’t actually write out the step shown on the second line in the example above. We usually just combine the steps of distributing into each term and simplifying, going directly from the first line to the third. From here on in we will not write out that step in this book, but whether or not you want to write that step out in your own work is entirely your choice.
    🔗
    The following example will illustrate a very important point about distributing when a subtraction is involved. Pay particular attention to this next example!
    🔗

    Example 2.2.3.

    Simplify \(4x(x-3)-3x(x+2)\)
    🔗
    🔗
    It is clear that we need to distribute \(4x\) into the first set of parentheses. In the second set, though, it is easy to make a mistake. It appears that we need to just distribute the \(3x\text{,}\) which would give us the following (incorrect) result:
    \begin{equation*} \text{\underline{Incorrect:} } 4x^2-12x-3x^2+6x \end{equation*}
    🔗
    The error here is that the “\(-\)” is only being applied to the first part of what’s inside those parentheses. The “\(x\)” got the negative, but the “\(+2\)” did not.
    🔗
    To treat this correctly, in theory what we should do is to rewrite \(4x(x-3)-3x(x+2)\) as \(4x(x-3)+(-3x)(x+2)\text{,}\) using the fact that subtracting is the same as adding a negative. Then we would distribute the \(-3x\) in and everything would get the negative. In practice, most people just remember that when you distribute, the minus gets distributed as well.
    🔗
    In this case, that means that we’d bring along the “\(-\)” and distribute a \(-3x\) into the \((x+2)\text{.}\) Whichever way you choose to look at it, the result is the same:
    \begin{equation*} \text{\underline{Correct:} } 4x(x-3)-3x(x+2)=4x^2-12x-3x^2-6x=x^2-18x \end{equation*}
    🔗
    Let’s try one additional example before moving on:
    🔗

    Example 2.2.4.

    Simplify \(5x(3x-1)-2(x^2-4x-3)-3(4x+2)\text{.}\)
    🔗
    🔗
    There are a lot of “\(-\)” ’s to deal with here. Take your time and be careful!
    \begin{align*} 5x(3x-1)\amp -2(x^2-4x-3)-3(4x+2)\\ \amp = 5x(3x-1)+(-2)(x^2-4x-3)+(-3)(4x+2)\\ \amp = 15x^2-5x-2x^2+8x+6-12x-6\\ \amp = 13x^2-9x \end{align*}
    🔗
    The notion of distribution extends from simple terms as demonstrated in the following examples:
    🔗

    Example 2.2.5.

    Simplify \((x+2)(x+3)\)
    🔗
    🔗
    We begin by distributing the second expression across the addition in the first:
    \begin{align*} (x+2)(x+3)\amp = x(x+3)+2(x+3)\\ \amp\\ \amp = x^2+3x+2x+6\\ \amp\\ \amp = x^2+5x+6 \end{align*}
    🔗

    Example 2.2.6.

    Simplify \((2x-4)(3x-1)\)
    🔗
    🔗
    We distribute and combine like terms:
    \begin{align*} (2x-4)(3x-1) \amp = 2x(3x-1)-4(3x-1)\\ \amp\\ \amp = 6x^2-2x-12x+4\\ \amp\\ \amp = 6x^2-14x+4 \end{align*}
    🔗

    Example 2.2.7.

    Simplify \((x-2)(2x^2+3x+4)\)
    🔗
    🔗
    We distribute and combine like terms:
    \begin{align*} (x-2)(2x^2+3x+4) \amp = x(2x^2+3x+4)-2(2x^2+3x+4)\\ \amp\\ \amp = 2x^3+3x^2+4x-4x^2-6x-8\\ \amp\\ \amp = 2x^3-x^2-2x-8 \end{align*}
    🔗
