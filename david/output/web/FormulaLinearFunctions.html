    Section 5.3 Finding Formulas for Linear Functions

    Of course not every function is linear, but lots and lots of useful functions are. We’ve already seen some of the properties of linear functions, specifically how to find their slopes and intercepts, and what those slopes and intercepts tell you.
    🔗
    In almost all of our work so far, though, we’ve assumed that we already had a formula for a given linear function, and then could work from that formula. In this section, we’ll fully explore something we’ve hinted at in a few examples but not yet developed: how to find the formula for a function that we’ve determined to be linear. As you’ll see, the properties of linear functions that we’ve already observed are not just helpful in interpreting functions whose formulas we somehow already know – they are incredibly useful in helping us find formulas for those functions in the first place!
    🔗

    Subsection 5.3.1 Formulas for Linear Functions Given Slope and Intercept

    In Chapter 5.2, we introduced the slope-intercept form for a linear function, \(f(x)=mx+b\text{.}\) We noted that a function is linear if and only if it can be written in this form. So, if we know the slope and vertical intercept of a linear function, we can write a formula for the function simply by plugging that slope and intercept into this general form.
    🔗

    Example 5.3.1.

    Suppose \(f(x)\) is a linear function whose slope is \(-5\) and vertical intercept is \((0, 8)\text{.}\) Find a formula for \(f(x)\text{.}\)
    🔗
    🔗
    Since it is linear, we know that \(f(x)=mx+b\) for the appropriate values of \(m\) and \(b\text{.}\) Now \(m\) is the slope, so we know that \(m\) is \(-5\text{.}\) The vertical intercept can be given as a point, as it is here, or just as the output value of that point which would be just \(8\text{,}\) and so that’s \(b\text{.}\) So substituting thosevalues into the slope-intercept form we get that
    \begin{equation*} f(x)=-5x+8 \end{equation*}
    🔗
    It really is just that simple, if we know the slope and vertical intercept. Of course, we won’t always have those values at our disposal, but since they are pretty significant features, it’s actually not all that unusual to have them available to find a linear function’s formula. In fact, when we worked out a couple formulas at the start of Section 5.2 using common sense reasoning, this is in fact what we were doing.
    🔗

    Example 5.3.2.

    A new bottle of vitamins contains \(75\) pills, and a full day’s dose is \(3\) pills. Find a formula for the number of pills the bottle will contain after \(t\) full daily doses have been taken.
    🔗
    🔗
    The function is linear, because with each full daily dose the number of pills changes by the same amount, it drops by three. The vertical intercept is the starting point, or more precisely, it is the number of pills in the bottle when the number of daily doses taken is zero. So \(m\) is \(-3\) and \(b\) is \(75\text{.}\) And so:
    \begin{equation*} f(t)=-3t+75 \end{equation*}
    🔗
    It’s not unusual to change the formula a bit, especially when a different order might seem to better reflect the “story” the function is describing. Here, we start with \(75\) pills and then take away \(3\) each day, so it might seem more natural to write this as \(f(t)=75-3t\text{.}\) Of course, this is algebraically equivalent to the formula we wrote above, so this alternative is also a correct formula for the function. At least for now, we will try to stick with the \(mx+b\) order, to emphasize that this is what we are using, but it is not at all incorrect to write this function either way.
    🔗
    🔗

    Subsection 5.3.2 Formulas for Linear Models Given Slope and Input-Output Pair

    Of course, we aren’t always lucky enough to have both the slope and the vertical intercept. If we have the slope and one input-output pair for a linear function, we can still find the function’s formula, albeit with a little more effort required. We will present two different methods for doing this by way of a specific example. Either method will always work, and so which one you use is a matter of preference.
    🔗

    Definition 5.3.3.

    Method #1: Back Substitution
    1. Start with the general form \(f(x)=mx+b\) (replacing the \(x\) and \(f(x)\) with the actual names of the function and input variable if they are different)
      🔗
      🔗
    2. Plug in the known value of the slope as \(m\text{.}\)
      🔗
      🔗
    3. Using the known input-output pair, plug in the input for \(x\) and the output for \(f(x)\)
      🔗
      🔗
    4. Solve the resulting equation for \(b\) to get the vertical intercept.
      🔗
      🔗
    5. You now have the slope and vertical intercept; use these in the “\(f(x)=mx+b\)” form to get the function formula.
      🔗
      🔗
    🔗
    🔗
    We can see how this works with the following two examples. We will actually start with a “numbers only” example just to show how this method works, and then will move to a real-world application example which will allow us to see both how the method works and a little bit of interpretation along the way.
    🔗

    Example 5.3.4.

    Find the formula for the linear function \(g(t)\) given that the function’s slope is \(-4.5\) and the function’s graph passes through the point \((5, 12)\text{.}\)
    🔗
    🔗
    1. We begin with the fact that since we know \(g(t)\) is linear, it must have a formula of the form:
      \begin{equation*} g(t)=mx+b \end{equation*}
      🔗
      🔗
    2. Now since we know the slope, we can plug that in to get:
      \begin{equation*} g(t)=-4.5t+b \end{equation*}
      🔗
      🔗
    3. Next, since the graph passes through the point \((5,12)\) we know that \(g(5)=12\text{,}\) and so if we plug in the input of \(5\) for \(t\) the output must be \(12\text{.}\) Plugging those values in we get:
      \begin{equation*} 12=-4.5(5)+b \end{equation*}
      🔗
      🔗
    4. Now this is an equation we can solve for \(b\text{.}\) Doing that we get:
      \begin{equation*} 12 = -22.5 + b \end{equation*}
      \begin{equation*} b=34.5 \end{equation*}
      🔗
      🔗
    5. So now we know that our function’s slope is \(-4.5\) and its vertical intercept is \(34.5\text{.}\) So, putting these facts together, we conclude that the formula is:
      \begin{equation*} g(t)=-4.5t+34.5 \end{equation*}
      🔗
      🔗
    In this first example we listed the numbered steps of the method explicitly. We won’t do that in future examples, but you can see that we are in fact still following them. In the next example, we’ll use the method to find a formula where there is a real world interpretation to the linear function and see how this gives some extra insight into why this method works.
    🔗

    Example 5.3.5.

    An online music site charges a fixed monthly membership fee, which allows unlimited streaming. Members can also download songs for \($0.85\) each. Last month I downloaded \(32\) songs and paid \($35.20\text{.}\) Find a formula for the amount I pay as a function of the number of songs I download.
    🔗
    🔗
    First we note that this function would indeed be linear. The input is music purchased (counted in songs), and the output is cost (in dollars). The rate of change would be dollars per song, and we are told this is a constant \($0.85\) per song. So the function is linear.
    🔗
    If we let \(s\) be the number of songs and \(C(s)\) be the cost for downloading that many songs, then:
    \begin{equation*} C(s)=mx+b \end{equation*}
    🔗
    Plugging in the known slope we get:
    \begin{equation*} C(s)=0.85s+b \end{equation*}
    🔗
    Since an input of \(32\) songs gives and output of a \($35.20\) charge, we know:
    \begin{equation*} 35.2 = 0.85(32)+b \end{equation*}
    🔗
    And so we solve for \(b\) and get:
    \begin{equation*} 35.2 = 27.2+b \end{equation*}
    \begin{equation*} b=8 \end{equation*}
    🔗
    Putting the slope and vertical intercept together we get that the overall formula must be:
    \begin{equation*} C(s)=0.85s+8 \end{equation*}
    🔗
    We can add some interpretation to this formula as well. The vertical intercept is the charge you’d pay when the input - the number of songs you download - is zero. So, we can interpret this function as telling us that you pay \($8\) per month just to have the service, plus \($0.85\) per download. }
    🔗
    This example agrees with a less-structured but commonsense way of figuring this out. Rather than using the formal method we’ve laid out here, you could have figured this formula out by reasoning that since each download costs \($0.85\text{,}\) I must have paid \(32($0.85)=$27.20\) for the downloads. Since my total charge was \($35.20\text{,}\) the membership fee must have been \($35.20-$27.20 = $8\text{.}\) From this you could have easily arrived at the formula.
    🔗
    Such reasoning is perfectly correct, and in fact if you look through the example, it’s actually pretty much what we did! There’s nothing wrong with taking this less-structured approach, but in cases where the interpretation is not so easy to see it may not work so well. The structured method we’ve laid out here always works, whether the interpretation is easy, difficult, or even if there is no interpretation at all (as in Example 5.3.4).
    🔗
    Now at the start of this section we promised a second method, and we will introduce that now, and then illustrate its use with the same two examples we’ve just worked out with the first method. Again, it must be emphasized that both methods will always work, but for a number of reasons, your instructor may prefer, or even insist that you use one or the other. Make sure that you are clear about the expectations in the course that you are taking.
    🔗
    🔗

    Subsection 5.3.3 Point-Slope Form

    This second method is a formula, derived from the formula for slope. Before presenting the formula, we’ll briefly show where it comes from. Though you need not fear being asked to reproduce the derivation yourself, it’s good to see at least once that our formulas really do come from somewhere!
    🔗
    Our formula for slope can be written in the form:
    \begin{equation*} m=\frac{\Delta f(x)}{\Delta x} = \frac{\Delta output}{\Delta input} \end{equation*}
    🔗
    Now, suppose we have a particular input-output pair for a linear function. If we call this particular input value \(x_0\) then we would call the output value \(f(x_0)\) or using an output variable we could call it just \(y_0\text{.}\) To keep the notation a little simpler, we will use \(y_0\) instead of the function notation here.
    🔗
    Now let’s let just plain \(x\) and just plain \(y\) be any other input and output for this function. (To be clear: \((x_0,y_0)\) is the specific input-output pair that we know, and just plain \((x,y)\) represents any other input-output pair for the function - not any specific input-output pair that we know.)
    🔗
    \(\Delta output\) and \(\Delta input\) can be found by subtracting outputs and inputs, and so going back to the slope formula we’d get:
    \begin{equation*} m=\frac{y-y_0}{x-x_0} \end{equation*}
    🔗
    Which we can then solve for \(y\) by first clearing the fraction and then simplifying:
    \begin{align*} \frac{y-y_0}{x-x_0}\amp =m\\ \left( \frac{y-y_0}{x-x_0} \right)(x-x_0) \amp = m(x-x_0)\\ y-y_0\amp =m(x-x_0)\\ y\amp =m(x-x_0)+y_0 \end{align*}
    🔗
    This work has led us to the point-slope form for the equation of a linear function. Plugging in the slope and known input-output pair directly into this form gives an equation for the linear function; it can then be readily simplified into slope-intercept form.
    🔗

    Definition 5.3.6.

    Method #2: Point-Slope Form
    🔗
    🔗
    If \(m\) is the slope and \((x_0,y_0)\) is a given input-output pair for a linear function, the function’s formula can be found by plugging these values into the form:
    \begin{equation*} y=m(x-x_0)+y_0 \end{equation*}
    🔗
    If the input variable is a letter other than \(x\) that letter would be used in place of \(x\) in this form; if there is an output variable given then that letter would likewise be used in place of \(y\text{.}\)
    🔗
    This can then be simplified into slope-intercept form if desired, and the \(y\) can be replaced with an \(f(x)\) to use function notation.
    🔗
    🔗
    We will illustrate the use of this method by reworking the previous two examples.
    🔗

    Example 5.3.7.

    Find the formula for the linear function \(g(t)\) given that the function’s slope is \(-4.5\) and the function’s graph passes through the point \((5,12)\text{.}\)
    🔗
    🔗
    Here the input variable is \(t\) and so we use that in place of \(x\text{.}\) There is no output variable given so we will use \(y\text{.}\)
    🔗
    We have here that \(m=-4.5\text{,}\) \(t_0=5\) and \(y_0=12\text{.}\) We plug these into the point-slope form to get:
    \begin{equation*} y=m(t-t_0)+y_0\\ y=-4.5(t-5)+12 \end{equation*}
    and now we simplify:
    \begin{equation*} y=-4.5t+22.5 + 12\\ y=-4.5t+34.5 \end{equation*}
    and finally rewriting in function form by replacing the \(y\) with an \(g(t)\text{:}\)
    \begin{equation*} g(t)=-4.5t+34.5 \end{equation*}
    🔗

    Example 5.3.8.

    An online music site charges a fixed monthly membership fee, which allows unlimited streaming. Members can also download songs for \($0.85\) each. Last month I downloaded \(32\) songs and paid \($35.20\text{.}\) Find a formula for the amount I pay as a function of the number of songs I download.
    🔗
    🔗
    Just as we did the first time, we must first note that this function would indeed be linear. The input is music purchased (counted in songs), and the output is cost (in dollars). The rate of change would be dollars per song, and we are told this is a constant \($0.85\) per song. So the function is linear.
    🔗
    If we let \(s\) be the number of songs and \(C(s)\) be the cost for downloading that many songs, as we did before, and let \(y\) stand in as the output variable, we have that \(m=0.85\) and the known input output pair \((s_0,y_0)\) is \((32,35.20)\text{.}\) Plugging this in to the point-slope form we get:
    \begin{gather*} y=m(s-s_0)+y_0\\ y=0.85(s-32)+35.2\\ y=0.85s-27.2+35.2\\ y=0.85s+8 \end{gather*}
    🔗
    Which we can then rewrite in function notation as:
    \begin{equation*} C(s)=0.85s+8 \end{equation*}
    🔗
    🔗

    Subsection 5.3.4 Formulas for Linear Models Given Two Input-Output Pairs

    Sometimes we are not even lucky enough to have the slope for a linear function. We can, however, still find the formula for a linear function if we have not just one but two input-output pairs for the function.
    🔗
    Remember that if we know two input-output pairs for a function, we can use them to calculate its slope. Then, once we have the slope, we can use the methods we’ve already discussed in this chapter to find the formula for the function. If we have the slope and one input-output pair we are set - and in this case, we actually have more than one!
    🔗

    Definition 5.3.9.

    Method for Finding the Formula for a Linear
    🔗
    Function Given Two Input-output Pairs
    1. Use the two input-output pairs to find the function’s slope
      🔗
      🔗
    2. Use the slope together with either one of the input-output pairs to find the function’s formula, using methods previously discussed.
      🔗
      🔗
    🔗
    🔗
    Essentially then we are just adding on an additional step at the start - finding the slope. Once we’ve done that, we are pretty much in the same boat as before.
    🔗

    Example 5.3.10.

    Find the formula for the linear function whose graph passes through the points \((2,9)\) and \((-1,24)\text{.}\)
    🔗
    🔗
    We first find the slope:
    \begin{align*} m\amp =\frac{\Delta output}{\Delta input}=\frac{\Delta f(x)}{\Delta x}=\frac{f(2)=f(-1)}{2-(-1)}\\ \amp =\frac{9-24}{2-(-1)}\\ \amp =\frac{-15}{3}\\ \amp =-5 \end{align*}
    🔗
    We now know the slope and can use it together with either of the two given input-output pairs. Here we will use \((2,9)\) for no particular reason other than we have to pick one of them. (Using the other pair will work equally well).
    🔗
    We’ll use the Back Substitution Method here (though using the point-slope form would work equally well. You’ve got choices!)
    🔗
    Since the function is linear we know it can be written in the form:
    \begin{equation*} f(x)=mx+b \end{equation*}
    🔗
    Plugging in the slope we get:
    \begin{equation*} f(x)=-5x+b \end{equation*}
    🔗
    And now plugging in the input-output pair and solving for \(b\text{:}\)
    \begin{align*} 9\amp =-5(2)+b\\ 9\amp =-10+b\\ b=19 \end{align*}
    🔗
    And finally putting this all together we get:
    \begin{equation*} f(x)=-5x+19 \end{equation*}
    🔗
    Knowing a second input-output pair gives an opportunity to check your answer if you like. If the formula is correct, we know it should give \(f(-1)=24\) because \((-1,24)\) is on its graph. You can verify for yourself that this does indeed work correctly.
    🔗

    Example 5.3.11.

    In patients with diabetes, blood sugar levels often fluctuate dramatically during the day and from day to day. Therefore, measuring a patient’s blood sugar level at any given time does not really reveal very much about overall blood sugar control.
    🔗
    A blood test known as “glycated hemoglobin” or “A1C” can be performed which measures a patients overall average blood sugar levels over the previous two to three months. It measures the percent of hemoglobin in the blood that has been affected by exposure to glucose. The average blood sugar level is a linear function of the A1C level. An A1C level of \(7.0\%\) corresponds to an average blood sugar of \(154 mg/dl\) (milligrams of glucose per deciliter of blood). An A1C level of \(10.0\%\) corresponds to an average blood sugar level of \(241 mg/dl\text{.}\) (Source: Wikipedia, “glycated hemoglobin”, accessed 7/24/11)
    1. Find a formula for average blood sugar level as a function of A1C.
      🔗
      🔗
    2. What would an A1C of 8.2 indicate for an average blood sugar?
      🔗
      🔗
    3. A typical average blood sugar for a non-diabetic would be 100. What A1C level would this correspond to?
      🔗
      🔗
    🔗
    🔗
    1. Here we know two input-output pairs, and so we can use them to find the slope:
      \begin{align*} m\amp =\frac{\Delta output}{\Delta input}=\frac{\Delta average \text{ } blood \text{ } sugar}{\Delta A1C}\\ \amp =\frac{241-154}{10-7}\\ \amp =\frac{87}{3}\\ \amp =29 \end{align*}
      Now we can use either input-output pair together with this slope to find the vertical intercept and complete the formula. Using \(x\) and \(f(x)\) for the input variable and function name we get:
      \begin{equation*} f(x)=mx+b \end{equation*}
      For no reason other than we’ve got to pick one, we’ll use the pair \((10,241)\text{:}\)
      \begin{align*} 241\amp =29(10)+b\\ 241\amp =290+b\\ b\amp =-49 \end{align*}
      Putting this together we get the formula:
      \begin{equation*} f(x)=-29x-49 \end{equation*}
      🔗
      🔗
    2. Now that we have the formula we can use it. Plugging in \(x=8.2\) we get:
      \begin{align*} f(x)\amp =29x-49\\ f(8.2)\amp =29(8.2)-49\\ f(8.2)\amp =188.8 \end{align*}
      So an A1C level of \(8.2\%\) indicates an average blood sugar level of \(188.8 mg/dl\text{.}\)
      🔗
      🔗
    3. Once again using the formula, but this time substituting in an output value:
      \begin{align*} f(x)\amp =29x-49\\ 100\amp =29x-49\\ 151\amp =29x\\ x=\frac{151}{29} \approx 5.21\text{.} \end{align*}
      So an average blood sugar of \(100 mg/dl\) equates to an A1C level of approximately \(5.21\%\text{.}\)
      🔗
      🔗
    🔗

    Linear functions by definition must have a constant rate of change. What this means in a given application, though, can lead to some confusion. For example, suppose that a small ice cream company is trying to mathematically model its profits. The company has fixed costs (that is, costs that they have to pay regardless of how much ice cream they sell) of \($4,000\) per month. They sell their ice cream for \($3.50\) per pint, and it costs them \($2.25\) to make each pint of ice cream. Ben says: “Well, our profit is clearly a linear function of the number of pints of ice cream we sell. Since \($3.50\) - \($2.25\) = \($1.25\text{,}\) we make the same, steady \($1.25\) per pint sold. Since our profits as a function of pints sold increase at a constant rate, the function is linear.”
    🔗
    Jerry says: “Nonsense! I agree that if our profit per pint sold were constant, the function would be linear, but it’s not! If we sell, say, \(5000\) pints, we take in \(5000($3.50) = $17,500\text{.}\) It costs us \(5000($2.25) = $11,250\) to make that ice cream, plus our fixed costs are \($4,000\text{,}\) for a total of \($15,250\text{.}\) So we make \($17,500 - $15,250 = $2,250\) profit. And that would be \($2,250/5000\) pints = \($0.45\) per pint. But now if we sell just \(2000\) pints, we take in \(2000($3.50) = $7,000\text{.}\) That ice cream costs \(2000($2.25) = $4,500\) to make, and so when you add in our fixed costs our costs are \($8,500\) in total. So our ``make” \($7,000 - $8,500 = -$1,500\text{.}\) So our “profit” is \(-$1,500/2000 = -$0.75\) per pint. A loss of \(75\) cents a pint sure ain’t the same as a profit of \($0.45\) per pint!’’
    🔗
    Who is right? Explain.
